## Overview

A facade system where each panel's geometry is determined by the intersection of solar radiation analysis, structural load paths, and CNC milling constraints. The result is a field of differentiated elements that reads as a coherent whole.

## Process

Environmental data was collected over twelve months and processed using Ladybug Tools inside Grasshopper. Panel geometries were then rationalized using a custom subdivision algorithm that respects both formal intent and manufacturing limits.

## Result

The system was prototyped at 1:5 scale using 3-axis CNC milling. Full installation comprises 340 unique panels across a 200m² facade surface.
